--help: lists all available commands
move <direction>: moves the player from room to room
pickup <item>: places an item in the user inventory
drop <item>: removes an item from the user inventory and places it in the current room
use <item>: uses an item in the player inventory
inventory: displays all the items in the player inventory
items: displays all the items in the current room
puzzle: displays the puzzle in the current room
